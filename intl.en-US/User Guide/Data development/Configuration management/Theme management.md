# Theme management {#concept_fyg_wkt_q2b .concept}

This topic is an overview of theme management. There are many tables in table management, where the tables are stored under the second-level sub-Folder according to the selected topics. These folders are summarized in the table, which is the theme. The administrator can add multiple themes based on project requirements, classify and organize the tables according to their purpose and name.
