# 接口数据格式-柱状图/折线图/饼图/雷达图 {#concept_spc_g3y_w2b .concept}

```
{
  "code": "200",
  "data": [
    {
      "name": "降雨量",
      "data": [
        {
          "name": "day1",
          "value": 30
        },
        {
          "name": "day2",
          "value": 20
        },
        {
          "name": "day3",
          "value": 15
        },
        {
          "name": "day4",
          "value": 40
        },
        {
          "name": "day5",
          "value": 31
        },
        {
          "name": "day6",
          "value": 26
        },
        {
          "name": "day7",
          "value": 17
        }
      ]
    }
  ],
}
```

**说明：** 

最外层数据为数组，里面包含1个对象。对象中包含name字段，和data数组。name表示图的系列名，data表示具体7天降雨量数据。
