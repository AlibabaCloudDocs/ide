# 固定列分割 {#concept_g14_yls_wgb .concept}

固定列分割，顾名思义，就是按照特定分割符来分割固定的列。

## 使用示例 {#section_rrz_5sf_xgb .section}

您可以使用固定列分割功能，将以下日志：

```
1111,2222,3333,4444
```

按照,分割为1111、2222、3333、4444四列。

固定列分割适用于列是按指定分隔符固定排列的日志。

|选择字段|选择要进行分割的字段名|
